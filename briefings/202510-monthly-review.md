**問題描述**：本月度回測 SKILL.md 使用 `curl` 或 `web_fetch` 直接抓 GitHub `data.json`，但：
1. 沙箱的 bash 環境透過 localhost:3128 Proxy，**封鎖了 raw.githubusercontent.com**
